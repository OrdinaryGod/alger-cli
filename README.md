# Alger CLI

## Documentation
这是一套前端工程脚手架，用于帮助快速创建主流前端工程。该脚手架主要支持以 webpack 和 vite 为构建工具的 Vue3+ 和 React18+ 工程。

## Installation

    $ npm install -g alger-cli
    # OR
    $ yarn global add alger-cli
    # OR
    $ pnpm add -g alger-cli

## Use
```bash
alger-cli create <project-name>

```
